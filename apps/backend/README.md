# Backend Application

This is a backend application focused on domain logic and perpetuation.

This project was bootstrapped with [NestJS CLI](https://docs.nestjs.com/cli/overview).

## Available Scripts

### `pnpm run dev`

Runs the app in the development mode with using [`nest start`](https://docs.nestjs.com/cli/usages#nest-start).
Open [http://localhost:3333](http://localhost:3333) to view it in your browser.

`--debug` option can run the app in the debug mode.

### `pnpm run build`

Creates an optimized production build of this application with using [`nest build`](https://docs.nestjs.com/cli/scripts#build).

### `pnpm run start`

Starts the application in production mode.
The application should be compiled with build first.

### `pnpm run lint`

Runs ESLint.

### `pnpm run test`

Launches the test runner for unit tests.

### `pnpm run test:debug`

Launches the test runner

### `pnpm run e2e`

Launches the test runner for e2e tests.
