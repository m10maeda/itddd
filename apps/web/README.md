# Web Frontend Application

This is a web frontend application focused on presentation.

This project was bootstrapped with [Create Next App](https://nextjs.org/docs/app/api-reference/create-next-app).

## Available Scripts

### `pnpm run dev`

Runs the app in the development mode with using [`next dev`](https://nextjs.org/docs/app/api-reference/next-cli#development).
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `pnpm run build`

Creates an optimized production build of this application with using [`next build`](https://nextjs.org/docs/app/api-reference/next-cli#build).

### `pnpm run start`

Starts the application in production mode with using [`next start`](https://nextjs.org/docs/app/api-reference/next-cli#production).
The application should be compiled with build first.

### `pnpm run lint`

Runs ESLint with using [`next lint`](https://nextjs.org/docs/app/api-reference/next-cli#lint).

### `pnpm run stylelint`

Runs Stylelint.

### `pnpm run test`

Launches the test runner.

### `pnpm run storybook`

Compiles and serves a development build of this project Storybook.

### `pnpm run storybook:build`

Compiles this project Storybook instance so it can be deployed.
