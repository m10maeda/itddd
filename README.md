# ITDDD

[![Project Status: Concept – Minimal or no implementation has been done yet, or the repository is only intended to be a limited example, demo, or proof-of-concept.](https://www.repostatus.org/badges/latest/concept.svg)](https://www.repostatus.org/#concept)

This project is a learning example of Domain Driven Design, based on a [Clean Architecture-aware example](https://github.com/nrslib/itddd#cleanlike) from Chapter 14 of "[ドメイン駆動設計入門 ボトムアップでわかる! ドメイン駆動設計の基本](https://www.amazon.co.jp/dp/479815072X)(Introduction to Domain Driven Design: The Basics of Domain Driven Design from the Bottom Up)".
