# ITDDD

This is an example designed and configured as a BFF(Backends For Frontends) architecture, based on [Clean Architecture-aware example](https://github.com/nrslib/itddd#cleanlike) in Chapter 14 of "[ドメイン駆動設計入門 ボトムアップでわかる! ドメイン駆動設計の基本](https://www.amazon.co.jp/dp/479815072X)(Introduction to Domain Driven Design: The Basics of Domain Driven Design from the Bottom Up)".
